Python Functions Cheatsheet

Function Description Example
print() Prints the specified message to the screen. print('Hello, World!')
len() Returns the length of an object. len([1, 2, 3, 4, 5])
type() Returns the type of an object. type(42)
range() Returns a sequence of numbers. list(range(5))
int() Returns an integer number. int('10')
str() Returns a string object. str(123)
float() Returns a floating point number. float('3.14')
list() Returns a list object. list('python')
dict() Returns a dictionary object. dict(name='Alice', age=25)
tuple() Returns a tuple object. tuple([1, 2, 3])
set() Returns a set object. set([1, 2, 3, 3, 4])
sorted() Returns a sorted list of elements. sorted([3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5])
sum() Returns the sum of all elements in the list. sum([1, 2, 3, 4, 5])
max() Returns the largest item in an iterable or the largest of two or more arguments. max([1, 2, 3, 4, 5])
min() Returns the smallest item in an iterable or the smallest of two or more arguments. min([1, 2, 3, 4, 5])
any() Returns True if any element of an iterable is true. any([False, False, True])
all() Returns True if all elements of an iterable are true. all([True, True, True])
input() Reads a line from input, converts it to a string (stripping a trailing newline), and returns that. input('Enter your name: ')
